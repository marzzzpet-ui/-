## 📖 О проекте

**АС «Астрономия»** — это десктопное приложение на C# (Windows Forms), разработанное в рамках курсовой работы по дисциплине «Технология разработки программного обеспечения». Проект предназначен для автоматизации рутинных астрономических вычислений, используемых в учебных и научно-исследовательских целях.

Главная цель — предоставить студентам и преподавателям интуитивно понятный инструмент для решения 7 типовых задач из сферической астрономии, небесной механики и астрофизики, заменив громоздкие ручные расчёты и электронные таблицы.

## ✨ Основные возможности

-   **Главное окно с тематическим дизайном:** Звёздное небо и силуэт наблюдателя погружают в рабочую атмосферу.
-   **Решение 7 астрономических задач:**
    -   Расчёт синодического периода планет (на примере Сатурна).
    -   *(Здесь вы можете кратко перечислить остальные 6 задач, которые у вас реализованы, например:)*
    -   Преобразование экваториальных координат в горизонтальные.
    -   Расчёт параметров орбиты по законам Кеплера.
    -   Учёт атмосферной рефракции.
    -   Вычисление звёздной величины.
-   **Интуитивный интерфейс:** Каждая задача открывается в отдельном окне с чёткой структурой «Условие → Дано → Решение → Ответ».
-   **Валидация данных:** Защита от некорректного ввода и деления на ноль.
-   **Поддержка русскоязычного интерфейса** и астрономической терминологии.
